# fsharp-lsp

A Claude Code plugin that provides F# language intelligence via [FsAutoComplete](https://github.com/fsharp/FsAutoComplete).

## Installation

**Install FsAutoComplete**

```bash
dotnet tool install -g fsautocomplete
```

**Install Plugin**

Ensure this marketplace is added to Claude (see marketplace [installation instructions](../../README.md#installation)), then:

```bash
claude plugin install fsharp-lsp
```
