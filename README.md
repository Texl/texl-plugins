# Claude Code Plugin Marketplace

A collection of [Claude Code](https://claude.ai/code) plugins.

## Plugins

| Plugin | Description |
|--------|-------------|
| [fsharp-lsp](plugins/fsharp-lsp/) | F# language intelligence via FsAutoComplete |

## Installation

**Install Marketplace**

```bash
claude plugin marketplace add https://github.com/Texl/texl-plugins.git
```
